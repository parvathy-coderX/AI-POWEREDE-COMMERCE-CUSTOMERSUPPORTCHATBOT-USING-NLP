# AI-Powered E-Commerce Customer Support Chatbot using NLP

## Project Overview

The **AI-Powered E-Commerce Customer Support Chatbot** is an intelligent chatbot designed to assist customers by answering common queries related to e-commerce services such as order tracking, shipping information, return policies, and payment methods. The chatbot uses **Natural Language Processing (NLP)** techniques to understand user queries and provide appropriate responses based on a predefined FAQ dataset.

This system helps automate customer support, reduce response time, and improve the overall user experience in online shopping platforms.

---

## Features

* AI-based chatbot for e-commerce customer support
* Answer Frequently Asked Questions (FAQ)
* Order tracking using order ID
* Natural Language Processing for understanding user queries
* Web-based chatbot interface
* Fast response using similarity-based matching

---

## Technologies Used

* **Python** – Backend programming
* **Flask** – Web framework for chatbot server
* **NLTK** – Natural Language Processing
* **Scikit-learn** – TF-IDF Vectorization and Cos
